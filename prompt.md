# Creation of the focal points feature

## The focal points feature

As a user, I want to see my master image and move a rectangle on it of the exact size for the current asset.
I want to be able to confirm my selection for all 14 focal points and be able to navigate next and previous between my assets.

## The focal points

The focal points should be combined in a single object to send the the server when the user confirms the 14 selection. 

## The focal points page

The focal points page should not break the brand of the application.

## Details

- Use the frontend design skills.
- Respect the current architecture of the application.
- Make this as modular as possible.
