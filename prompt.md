# Creation of the upload feature

## The upload feature

As I user, when I am on the landing page, I want to click on a get started button and be redirected to the upload page.

## the landing page

You need to update the current CTA that invites to join the waitlist so it offer the user to Get Started.
Make sure this CTA is inviting and show the value proposition.

## The upload page

The upload page should do the following:

- Create a session on the server and store it in local storage
- Get a signed URL for the upload
- Have a zone to upload the file (I dont care about drag and drop for now)

## Details

- Use the frontend desing skills.
- Create a utility function to connect to the api that handle errors and return just the data.
- Use the `fetch` api to get the data from the server.
